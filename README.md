# LIKO-12 Player
LIKO-12 Player es un proyecto desarrollado con LÖVE (love2d), al igual que LIKO-12, este proyecto es una modificación de LIKO-12 para coger todo este código, añadir tu .lk12 con el nombre "game.lk12", lo comprimes, renombras a .love y ejecutas, tendrás un juego que contendrá el motor, este copiará tu juego, lo pondrá en el disco D virtual, lo cargará con load, y lo ejecutará con run, y listo, tendrás tu juego corriendo rápidamente solo con abrirlo, no requiere hacerlo a mano.

# Compilar
Coge el .love y el love.exe de donde descargaste LÖVE (love2d), abre CMD y ejecuta este commando:
cmd ´´
copy /b love.exe + nombre_del_love.love nombre_del_exe_compilado.exe
´´
Luego, coge todos los .dlls y el llamado license.txt, cópialos junto al EXE, y renombra el license.txt a cualquier nombre que de a entender que es de LÖVE y no tuyo (ej. LOVE-LICENSES.txt), con solo dejar el TXT cumplirás con las licencias de terceros.
Ahora, publica el juego comprimiéndolo en el formato que gustes.
